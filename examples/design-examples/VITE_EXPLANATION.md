# Vite란?

## 🚀 Vite가 무엇인가요?

**Vite**는 빠른 프론트엔드 빌드 도구입니다. 

### 왜 Vite를 사용하나요?

1. **매우 빠른 개발 서버**
   - 전통적인 도구(Webpack 등)는 프로젝트가 커질수록 시작이 느려집니다
   - Vite는 **ESM(ES Modules)**을 직접 사용하여 즉시 시작합니다
   - 코드 변경 시 **HMR(Hot Module Replacement)**이 매우 빠릅니다

2. **최적화된 프로덕션 빌드**
   - Rollup 기반으로 번들링하여 최적화된 결과물을 생성합니다
   - 코드 스플리팅, 트리 쉐이킹 등이 자동으로 적용됩니다

3. **TypeScript, React 등 즉시 지원**
   - 추가 설정 없이 TypeScript와 React를 바로 사용할 수 있습니다
   - `@vitejs/plugin-react` 플러그인으로 JSX 변환을 처리합니다

### 이 프로젝트에서 Vite를 사용하는 이유

예제 갤러리는 **빠른 개발과 미리보기**가 중요합니다:
- 디자이너가 컴포넌트를 빠르게 확인할 수 있어야 합니다
- 코드 변경 시 즉시 반영되어야 합니다
- 빌드 시간이 짧아야 합니다

### Vite vs 다른 도구

| 도구 | 시작 속도 | HMR 속도 | 설정 복잡도 |
|------|----------|----------|------------|
| **Vite** | ⚡ 매우 빠름 | ⚡ 매우 빠름 | ✅ 간단 |
| Webpack | 🐌 느림 | 🐌 느림 | ❌ 복잡 |
| Create React App | 🐌 느림 | 🐌 느림 | ✅ 간단 |

### 실행 방법

```bash
# 개발 서버 시작
npm run dev

# 프로덕션 빌드
npm run build

# 빌드 결과 미리보기
npm run preview
```

개발 서버는 보통 `http://localhost:5174`에서 실행됩니다.
